N.P.D.O
Non-Profit Data Organizer

Program will take the EIN of a non profit organization and search for the financial data spanning the years that were recorded
on Propublica.

Three files are needed to run the program. 
Webscraper.py, fine.py, and mainGUI.py should all be within the same folder. Only fine.py needs to be opened and ran.

1. Open fine.py in IDE
2. Run fine.py
3. Window to input EIN should show up
4. Find an EIN for a non profit organization and input into EIN window.
